# Change Log

All notable changes to the "vscode-task-mcp" extension.

## [0.1.0] - 2025-05-10

### Added

- Initial release
- `list_tasks` MCP tool: list all VSCode tasks in the current window
- `execute_task` MCP tool: execute a task by name and wait for completion
- Multi-window support via PID-based port discovery
- MCP stdio bridge (`mcp-stdio-bridge.js`) for Cline integration
